What Is the Number Memory Test?

The number memory test displays a sequence of digits for a fixed time, then asks you to recall them in order. It measures your working memory capacity — how many items you can hold in active conscious memory at one time. This is one of the most fundamental cognitive measures used in psychology and neuroscience research.

Each correct round increases the digit count by one. The test ends when you make an incorrect response. Your final score (digit count) is your working memory span for numerical information.

The average working memory span is 7 ± 2 digits, a finding established by psychologist George Miller in 1956 and known as "Miller's Law." Modern research suggests the true capacity is closer to 4 independent "chunks" — but chunking strategies allow each chunk to contain multiple items.

Memory Improvement Techniques

  • Chunking: Group digits into 2–3 digit groups (e.g., 847291 → 847 · 291). Reduces 6 items to 2 chunks, well within working memory limits.
  • Verbal rehearsal: Silently repeat the number sequence during the recall phase. Articulatory loop in working memory benefits from verbal rehearsal.
  • Spaced practice: Daily testing yields measurable working memory improvements over 4–6 weeks.
