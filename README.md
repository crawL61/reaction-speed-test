# ⚡ Reaction Speed Test

Free online reaction time test, CPS test, typing speed test, 
aim trainer, memory test and browser games.

🌐 **Live site:** https://reactionspeedtest.com

## Features
- ⚡ Reaction Time Test (milliseconds)
- 🖱️ Click Speed Test (CPS) — 8 durations
- 📝 Typing Speed Test (WPM)
- 🎯 Stimulation Clicker (Aim Trainer)
- 🧠 Number Memory Test
- 💰 Clicker Tycoon (Idle Game)
- 🦕 Dinosaur Game, 🐍 Snake, 🎮 Coreball

## Tech Stack
Pure HTML/CSS/JavaScript — no frameworks, no dependencies.
Single-page app + 11 static content pages.
Hosted on Vercel.
